About Me

Hi everyone! My name is Mike Berger and I am a full-stack developer. After graduating from Rice University I spent 2.5 years working in Quality Assurance at a large tech firm in Wisconsin (called Epic). I tested Epic's Cogito Data Warehouse application and learned a ton about testing, databases and the software development cycle. After Epic I committed myself full time to learning web development by taking a 12 week immersive coding bootcamp in Silicon Valley. I graduated with certifications (they call them 'belts') in each stack offered: CSS + HTML, LAMP, MEAN and Ruby on Rails and gained a good 1000+ hours of coding experience. Now I am searching for my next opportunity.

Here is an app to create a single elimination tournament (made from scratch with Ember and Rails). It calculates number of byes necessary, each player's ELO rating for a win/loss (if Ping Pong is selected) and seeds based on rating or random. A user can drag and drop the winner to advance them in the tournament, or reverse a decision with an 'undo' button. About this site

This site was made using Materialize. Parallax looks best on Chrome and Firefox on Mac OS or Firefox on Windows. Is mobile-phone friendly. Spanish/English translation made with a javascript function saved for English and Spanish, original values in English stored in a closure, Spanish values stored in a separate file, elements get updated with DocumentGetElementById.
